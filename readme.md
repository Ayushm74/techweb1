# Futuristic site

Website made in futuristic style for no existing revolutionary auth technology

## Run Locally

Clone the project

```bash
  git clone https://github.com/Pla9er9/Futuristic-site
```

Go to the project directory

```bash
  cd Futuristic-site
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
